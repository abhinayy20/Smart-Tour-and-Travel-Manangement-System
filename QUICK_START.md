## 🚀 How to Open

### Option 1: Direct File
Simply double-click `index.html` to open in your default browser.

### Option 2: Local Server (Recommended)
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Then visit: http://localhost:8000
```

### Option 3: VS Code Live Server
1. Install Live Server extension in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

## 🎨 Customization

### Change Logo/Brand Name
Edit the navbar section in `index.html`:
```html
<div class="logo">
    <i class="fas fa-globe"></i>
    <span>TravelSphere</span>  <!-- Change this -->
</div>
```

### Update Contact Information
Find the contact section and update:
- Phone numbers
- Email addresses
- Office address
- Working hours

### Modify Tour Packages
Edit the packages section to add/remove packages. Each package needs:
- Package image background (gradient)
- Discount percentage
- Package name
- Duration
- Locations
- Price

### Change Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #667eea;      /* Main color */
    --secondary-color: #764ba2;    /* Accent color */
    --accent-color: #f5576c;       /* Highlight color */
}
```

### Add Social Media Links
Update footer social links:
```html
<a href="https://facebook.com/your-page"><i class="fab fa-facebook"></i></a>
<a href="https://instagram.com/your-page"><i class="fab fa-instagram"></i></a>
```
